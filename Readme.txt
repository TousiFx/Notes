# Notes App

A simple web application for taking and managing notes.

## Features

- Create new notes
- Delete notes
- Edit notes
- Store notes locally

## Technologies Used

- HTML
- CSS
- JavaScript

## Installation

1. Clone the repository:


2. Open the project directory:



3. Open `index.html` in your web browser.

## Usage

1. Click the "Create Note" button to create a new note.
2. Click on a note to edit its content.
3. Click the delete icon to delete a note.
4. Your notes are automatically saved locally.
